# **App Name**: AI Code Alchemist

## Core Features:

- Advanced Code Editor: A fully featured code editor with syntax highlighting for JavaScript, auto-indentation, line numbering, and bracket matching. Utilizes a library like CodeMirror or Ace Editor to provide a robust coding environment.
- AI Question Submission: Allows users to type in and submit questions related to their code directly within the editor.
- AI-Powered Assistance Tool: Uses a Large Language Model (LLM) to provide context-aware suggestions, explanations, and debugging assistance based on the code and question asked. This tool uses reasoning to decide whether code is complete or partial before responding.
- Response Display: Displays AI-generated responses clearly and concisely within the editor, possibly in a dedicated panel or overlaid window. It also can allow copying back suggested edits.

## Style Guidelines:

- Primary color: Light grey-blue (#B0C4DE), providing a calm coding atmosphere reminiscent of clear skies, helping to keep the focus on the code.
- Background color: Off-white (#F5F5F5), for a comfortable contrast that reduces eye strain during prolonged use.
- Accent color: Soft lavender (#E6E6FA), subtly highlighting key interactive elements such as buttons or active panels, and standing out from the primary hue without being distracting.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, and 'Inter' (sans-serif) for body text and code.
- Use minimalist icons for editor functions (save, run, debug) that are easily recognizable and don't clutter the interface.
- Divide the interface into distinct panels for the code editor, AI response display, and potentially a file explorer. Use a clean, grid-based layout to maintain organization.
- Use subtle transitions for AI responses appearing or code auto-completion suggestions. This enhances user experience without being distracting.
